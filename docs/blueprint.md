# **App Name**: Buzena

## Core Features:

- User Account & Role Management: Secure authentication (login, registration) for Clients, Deliverers, and Company Admins, with appropriate role-based access.
- Deliverer Profile & Verification Flow: Deliverers can submit required personal and vehicle documentation (selfie, ID, vehicle type); Global Admins can review and approve profiles to activate 'buzz' capabilities.
- Company Order Creation & Management: Company administrators can create and manage new delivery orders, including specifying client details and delivery addresses.
- Deliverer Job Board & Details: Deliverers can view a list of available and assigned orders, accept jobs, and access all necessary delivery information for successful completion.
- Digital Horn Trigger & Event Logging: Approved deliverers can initiate a 'buzz' notification via a prominent in-app button, logging the event in the database for tracking.
- Client Real-time Arrival Notification: Clients receive immediate, distinct sound and strong vibration notifications, along with basic deliverer details, when a deliverer triggers the 'buzz'.
- Post-Delivery Rating System: Clients can provide a rating for deliverers after a delivery is completed, contributing to the deliverer's reputation score.

## Style Guidelines:

- Primary accent color: A vibrant, attention-grabbing yellow (#FFD600) used for calls-to-action, alerts, and key interactive elements like the 'Buzinar' button.
- Main background colors: Deep black (#000) for a striking, modern backdrop, and clean white (#FFF) for contrast in content areas and improved readability.
- Accent color: A dynamic and energetic red-orange (#D84B0D) to provide secondary emphasis and signal important but less primary interactions.
- Body and headline font: 'Inter' (sans-serif) for its modern, neutral, and highly readable characteristics, suitable for conveying information across all user interfaces.
- Use a clear and intuitive 'horn' icon for the main buzzer button. Other icons should follow a minimalistic, geometric, or line-based style for consistency and clarity.
- Mobile-first, minimalistic design emphasizing large, circular action buttons (like the 'Buzinar' button) and clean, structured layouts for order lists and user profiles.
- Incorporate a noticeable but smooth animation when the 'Buzinar' button is pressed to provide immediate user feedback. Other interactions should have subtle, purposeful animations.
